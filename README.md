Password Strength Checker:

A modern GUI-based Password Strength Checker built using Python and CustomTkinter.

Features

- Password strength analysis
- Dynamic strength progress bar
- Colour-changing security indicator
- Common password detection
- Password entropy calculation
- Show/Hide password
- Modern dark-themed GUI
         		
Technologies Used:

- Python
- CustomTkinter
- Regex
- Cybersecurity concepts


How To Run

Install dependencies:

bash
pip install customtkinter

Run the application:

bash
python app.py
